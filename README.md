# My Retro portfolio

Local replica of [digital-maturation-portfolio](https://github.com/JVK0804/digital-maturation-portfolio), renamed for this copy.

Vite + React + TypeScript + Tailwind.

**Design system:** `design-system.md` (same spec as the main portfolio) drives tokens in `src/styles/tokens.css`, which is imported from `src/index.css`. Tailwind theme extensions live in `tailwind.config.ts` (`heritage`, `teal`, `ink`, serif / mono font families).

```bash
npm install
npm run dev
```
